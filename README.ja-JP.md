## 機能紹介

- 🎉 Tauri v2 をもとに開発、軽量で効率的、クロスプラットフォームの体験をさらに向上させる。
- 💻 Windows、macOS、Linux（x11）とも対応でき、複数デバイス間でシームレスに切り替え可能である。
- ✨ シンプルで直感的なユーザーインターフェース、敷居が低く簡単に利用可能になる。
- 📋 テキスト、リッチテキスト、HTML、画像、ファイル形式のクリップボード内容をサポートできる。
- 🔒 データはローカルに保存され、ユーザーのプライバシーを保護、データはユーザー自身が完全に管理できる。
- 📝 メモ機能をサポートでき、簡単に分類、管理、検索が可能で作業効率を向上させる。
- ⚙️ 豊富なカスタマイズ設定で、異なるユーザーのニーズを満たす個別体験を提供できる。
- 🤝 完善なドキュメントとコミュニティ機能をサポート、開発者と共に成長を目指す。
- 🧩 継続的な最適化し、もっと驚きの機能があなたの発見を待っている。

## 🚀 Fork 版の拡張機能

> 本リポジトリは [EcoPasteHub/EcoPaste](https://github.com/EcoPasteHub/EcoPaste) の Fork 版であり、オリジナル版に基づき以下のユーザビリティ改善を追加しています：

### ✨ 独自のスーパー拡張機能

- **独立グループと色認識**: 「Links」、「Colors」、「Code」などの原生専用グループ分類を新規追加。RGB/RGBAなどの色フォーマットを正確に抽出し強調表示。
- **充実した二次没入型編集**: 画像などのリッチコンテンツデータ向けの独立した二次モーダル編集と、システムレベルの高速ファイルロケーションに対応。
- **コードシンタックスハイライトプレビュー**: コピーされたコードスニペットを自動検出し、IDEに匹敵するネイティブ構文ハイライトレンダリングを提供。
- **データソース取得と表示容量拡張**: コピー元のアプリのアイコンとアプリケーション名を取得する原生ツールチップボックスを提供。各種設定の表示能力上限を拡張（1〜50行まで設定をサポート）。
- **システムレベルのスクリーンショット完全ダンプ**: SQLiteの履歴動作をインターセプトし再構築。画像をTauriエンジンの制限なしに指定した「カスタムされたドライブパス」に完全に保存し、Cドライブの無駄な消費を防止します。

### 📏 高度なテキストと画像の表示適合

- **テキストの行数構成カスタマイズ**: 設定 → クリップボード → 表示設定 → テキスト/コード/ファイルの表示行数（1〜50行）
- **画像プレビューのスケーリング制限**: 画像の高さの柔軟な調整をサポートし、長さに応じてインテリジェントに内容を展開 / 折りたたみます（50〜500ピクセル）。

### 🔄 動的表示機能とノン・ディスターブ没入フレーム

- **フルデータの展開**: コンテンツが表示制限を超えた場合、折りたたみ/展開ボタンを提供します。仮想スクロールにより状態が破壊されることはなく、状態は常に維持されます。
- **フォーカスを奪わない静音ウインドウ体験**(Windows): 呼び出しキーを使用すると、元のアプリにフォーカスを戻し、ダブルクリックでシームレスに直接貼り付け、元のアプリ側へ隠れたまま進行します。
- **自動入力カーソル追跡**: ウインドウがエディターのカーソル位置に応じて自動的に表示され、完璧なフロープロセスを実現。
- **設定アーキテクチャの再構築**: 「表示設定」グループを新しく分割し、「表示を折りたたむ」や「元コンテンツ」などの設定類を細分化。

### 🔧 データの無損失永続化メカニズム

- 全ての新規パラメータ、データファイル、およびスクリーンショットの追跡操作をオリジナル版から完全に分離し、高権限ディレクトリへ自動保存するため再インストール時にも保持されます。

### 🔄 自動連携と完全性

- このリポジトリは、毎日自動的に上流のEcoPasteHub/EcoPasteアップデートを確認します。
- レベルでのマージ統合による競合もIssue機能により完璧に捕捉します。

## 🐛 オリジナル版の完全修正計画

> オリジナル版内の深層のシステムクラッシュやエラーに対するバグ修正:

### 📋 コンテンツタイプ評価の破損と崩壊解決

- **問題の説明**: Web画像はHTMLのウェイトによりウェブページとして検知されやすく、ローカルスクリーンショットは「fullName」の問題により自己生成ディレクトリで広範囲に画像がクラッシュ表示される致命的なバグが存在します。
- **解決策**: 自動優先権チェックシステムを大幅改修し、画像を最優先に保護します。FSアーキテクチャパスのルートマッピングを再構築し、ディレクトリを安全に自動生成しながら正しいUIのURL表示プロトコルを完全に復元しました。

### 💾 自動バックアップの回復

- **問題の説明**: 権限の競合の制限で隠され消えたデータバックアップポート。
- **解決策**: デッドゾーンのバグを修復し入り口を正常に開放。データベースアーカイブの安全なエクスポート機能がシームレスに操作できるようになります。

### 🔧 設定の永続化

- すべての新規設定項目はユーザーデータディレクトリに自動保存されます
- アプリ更新後も設定は失われません
- 設定の手動エクスポート/インポートは不要です

### 🔄 アップストリームの自動同期

- 本リポジトリは毎日自動的にアップストリームの EcoPasteHub/EcoPaste の更新をチェックします
- 新しいバージョンがある場合、自動的にマージしてビルドをトリガーします
- マージ競合が発生した場合は、手動対応のために Issue を作成して通知します

## 🐛 オリジナル版の修正

> 以下はオリジナル版 EcoPaste の既知の問題に対する修正です：

### 📋 クリップボードコンテンツタイプ判定ロジック

- **問題の説明**：
  - ネット上の画像をコピーした際、アプリが「画像」タイプではなく「HTML」タイプとして誤認する場合がある
  - スクリーンショット後、アプリがスクリーンショット画像を「画像」タイプではなく「ファイル」タイプとして誤認する場合がある
- **修正案**：判定ロジックを修正し、コピー内容に画像データが含まれる場合、「画像」タイプとしての識別を優先するようにしました

### 💾 データバックアップ入り口

- **問題の説明**：データバックアップ機能の入り口が欠落している
- **修正案**：データバックアップと復元機能が正常に利用できるように、データバックアップ機能の入り口を表示しました
