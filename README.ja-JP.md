## 機能紹介

- 🎉 Tauri v2 をもとに開発、軽量で効率的、クロスプラットフォームの体験をさらに向上させる。
- 💻 Windows、macOS、Linux（x11）とも対応でき、複数デバイス間でシームレスに切り替え可能である。
- ✨ シンプルで直感的なユーザーインターフェース、敷居が低く簡単に利用可能になる。
- 📋 テキスト、リッチテキスト、HTML、画像、ファイル形式のクリップボード内容をサポートできる。
- 🔒 データはローカルに保存され、ユーザーのプライバシーを保護、データはユーザー自身が完全に管理できる。
- 📝 メモ機能をサポートでき、簡単に分類、管理、検索が可能で作業効率を向上させる。
- ⚙️ 豊富なカスタマイズ設定で、異なるユーザーのニーズを満たす個別体験を提供できる。
- 🤝 完善なドキュメントとコミュニティ機能をサポート、開発者と共に成長を目指す。
- 🧩 継続的な最適化し、もっと驚きの機能があなたの発見を待っている。

## 🚀 Fork 版の拡張機能

> 本リポジトリは [EcoPasteHub/EcoPaste](https://github.com/EcoPasteHub/EcoPaste) の Fork 版であり、オリジナル版に基づき以下のユーザビリティ改善を追加しています：

### ✨ 新機能

- **☁️ WebDAV クラウドバックアップ**：WebDAV プロトコルを使用してクラウドストレージ（Nutstore、NextCloud 等）にクリップボードデータをバックアップ。スリムバックアップモード（画像とファイルをスキップし、テキスト等のコンテンツのみ保持）、自動スケジュールバックアップ、バックアップ数制限、ワンクリック復元をサポート。
- **🎨 独立グループと色認識**：「リンク（Link）」、「色（Color）」、「コード（Code）」、「メール（Email）」などのネイティブ専用グループ分類を追加。RGB/RGBA 等の色フォーマットを正確に抽出し強調表示。
- **📝 充実した二次没入型編集**：テキスト等のリッチコンテンツの独立モーダル編集と、システムレベルの高速ファイルロケーションに対応。
- **💻 コードシンタックスハイライト**：コピーされたコードスニペットを自動検出し、IDE レベルの構文ハイライトレンダリングを提供（設定 → クリップボード → 表示設定 → コードの構文ハイライト）。
- **📊 ソースアプリ追跡と表示**：コピー元アプリのアイコンとアプリ名を提供（設定 → クリップボード → 表示設定 → アプリのソースを記録）。
- **⚡️ ネイティブクイックアクセス**：ファイルパスをエクスプローラーで直接開き、Webリンクをブラウザでワンクリックで開き、システムのデフォルト画像ビューアで画像を直接閲覧することをサポートします。
- **🗑️ スマート削除確認**：画像コンテンツを削除する際、確認ダイアログに「ローカルファイルも削除する」チェックボックス（デフォルト選択済み）を表示。ローカルファイルを保持したまま記録のみ削除可能。

### 📏 高度なテキスト・画像表示適合

- **テキスト/コード/ファイルの表示行数カスタマイズ**：設定 → クリップボード → 表示設定 → テキスト/コード/ファイル表示行数（1〜50 行）
- **画像高さスケーリング**：画像の高さを柔軟に調整、スマート展開/折りたたみ対応（50〜500 ピクセル）

### 🔄 動的展開/折りたたみと没入体験

- **フルデータ展開**：コンテンツが表示制限を超えた場合、展開/折りたたみボタンを提供。仮想スクロールにより状態が破壊されず常に維持。
- **フォーカスを奪わない静音ウィンドウ**（Windows）：元のアプリがフォーカスを保持、ダブルクリックでサイレント貼り付け、ウィンドウ外クリックで自動非表示。
- **入力カーソル追従**：エディターのカーソル位置に応じてウィンドウが自動表示。
- **設定アーキテクチャ再設計**：「表示設定」グループを追加し、高度なオプションを細分化。

### 🐛 オリジナル版バグ修正

- **📸 スクリーンショットの完全保存**：SQLite の永続化動作および基盤となる FS マッピングパスを再構築。オリジナル版の組み込みライブラリの制限により生じていた、カスタムディレクトリでのクラッシュや画像が適切に表示されない問題を完全に解決。ユーザー指定のカスタムディレクトリにスクリーンショット画像を保存でき、Cドライブの容量浪費を防止。
- **📋 クリップボードタイプ誤判定**：Web 画像が `html` 優先度により誤ってウェブページとして検出される問題を完全に修正。検出ロジックを書き直し、画像タイプを最優先にするように変更。
- **🔗 複製リンクの重複記録**：リンクをコピーした際、クリップボード上に同一の記録が二重に生成される問題を完全に修正。
- **💾 バックアップ入口復旧**：権限制限で隠されたデータバックアップ/復元入口を再有効化し、安定したデータインポートとエクスポート動作を保証。
- **⚙️ 設定の永続化**：すべての新規設定項目はユーザーデータディレクトリに自動保存。アプリ更新後も設定は失われません。

### 🔄 アップストリーム自動同期

- 毎日自動的に上流の EcoPasteHub/EcoPaste の更新をチェック
- 新バージョン検出時に自動マージしてビルドをトリガー
- マージ競合発生時は Issue を作成して通知
