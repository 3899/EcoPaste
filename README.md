## 功能介绍

- 🎉 基于 Tauri v2 开发，轻量高效，跨平台体验更进一步。
- 💻 支持 Windows、macOS 和 Linux（x11），在多设备间无缝切换。
- ✨ 简洁直观的用户界面，操作简单，零门槛上手，开箱即用。
- 📋 支持纯文本、富文本、HTML、图片和文件类型的剪贴板内容。
- 🔒 数据本地存储，确保用户隐私安全，数据完全掌控在用户手中。
- 📝 支持备注功能，轻松分类、管理和检索，让工作更高效。
- ⚙️ 丰富的个性化设置，满足不同用户需求，打造专属体验。
- 🤝 完善的文档与社区支持，与开发者共同探索与成长。
- 🧩 持续优化中，更多惊喜功能等你发现。

## 🚀 Fork 版本增强功能

> 本仓库是 [EcoPasteHub/EcoPaste](https://github.com/EcoPasteHub/EcoPaste) 的 Fork 版本，在原版基础上增加了以下易用性改进：

### 📏 文本显示行数配置

- **功能说明**：可自定义剪贴板条目的文本显示行数
- **配置位置**：偏好设置 → 剪贴板 → 内容设置 → 文本显示行数
- **可选范围**：1 - 20 行（默认 4 行）
- **效果**：
  - 内容行数 ≤ 设定值时，按实际行数显示
  - 内容行数 > 设定值时，按设定行数截断显示

### 🖼️ 图片显示高度配置

- **功能说明**：可自定义剪贴板中图片预览的最大高度
- **配置位置**：偏好设置 → 剪贴板 → 内容设置 → 图片显示高度
- **可选范围**：50 - 500 像素（默认 100 像素）
- **效果**：图片预览高度不超过设定值，保持宽高比缩放显示

### 🔄 展开/收起功能

- **功能说明**：当内容超过显示限制时，提供展开/收起按钮
- **适用范围**：
  - 文本内容：超过设定行数时显示展开按钮
  - 图片内容：超过设定高度时显示展开按钮
- **操作方式**：点击条目底部的展开/收起按钮切换显示状态

### 📍 跟随光标窗口位置（Windows）

- **功能说明**：剪贴板窗口可跟随输入光标位置显示
- **配置位置**：偏好设置 → 剪贴板 → 窗口设置 → 窗口位置 → 跟随光标
- **工作原理**：
  1. 优先获取当前输入框中的文本光标位置
  2. 如无法获取（如非文本输入场景），则回退到跟随鼠标位置
- **平台支持**：目前仅支持 Windows 系统
- **使用场景**：适合在文档编辑、代码编写等需要频繁粘贴的场景使用

### ⚙️ 配置持久化

- 所有新增配置项均自动保存到用户数据目录
- 应用更新后配置不会丢失
- 无需手动导出/导入配置

### 🔄 自动同步上游

- 本仓库每天自动检查上游 EcoPasteHub/EcoPaste 的更新
- 有新版本时自动合并并触发构建
- 如遇合并冲突会创建 Issue 通知手动处理
